# Menu

Related component
- [MenuItem](https://huynhhuyhiep.github.io/doopage-react-ui-kit/menu-item)

Icon
- https://material-ui.com/components/material-icons/

## Basic usage

```jsx
import { Menu } from '@doopage/react-ui-kit'

const Example = () => {
    return (
      <Menu
        label={'menu'}
        options={[{id: '1', name: 'option1', onClick: () => {console.log('on click')}}]}
      />
    );
}

export default Example;
```
