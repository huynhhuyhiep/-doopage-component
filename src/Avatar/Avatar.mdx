# Avatar
Base on https://material-ui.com/components/avatars/

Related component
- https://material-ui.com/components/badges/
- https://material-ui.com/components/tooltips/

## Basic usage

```jsx
import { Avatar } from '@doopage/react-ui-kit'

const Example = () => {
    return <Avatar src={'/local/image'} alt={'Image'}/>;
}

export default Example;
```
